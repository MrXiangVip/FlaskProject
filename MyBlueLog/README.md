# Bluelog

*A blue blog.*


## Installation


generate fake data then run:
```
$ 初始化db:     flask db init
$ 更新db:       flask db migrate -m "注释"
$ 应用更新:      flask db upgrade 
$ 生成虚拟数据:   flask forge
$ 运行 在pycharm 中运行或者执行 flask run  
* Running on http://127.0.0.1:5000/
```
